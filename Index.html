Projects

Digital Thermometer and Cooling System - Programming a Microcontroller in C

(Academic Project) Fall 2020

Built a digital thermometer circuit which interfaced with a microcontroller to display the current temperature and the user selected setpoint temperature on an LCD display. If the temperature exceeded the runtime changeable setpoint, a fan, whose circuit interfaced with the microcontroller, turned on to cool the sensor at a speed relative to the temperature difference.

BMade use of ADC channel to receive temperature sensor output and make digital conversion.

Made use of PWM timer to control fan speed.

Safety Data Sheet Online Library (Professional Project) Summer 2019

Made use of the free online cloud storage provided by google drive to create a high-quality safety data sheet repository.

Prosthetic Hand (Academic Project) Fall 2018

Designed a 3 dimensional two-finger prosthetic hand and geartrain using Autodesk Inventor under strict size restrictions.

Each component was then exported to Slic3r and 3D printed.

These components were then sanded, lubed, and assembled to create a functional prosthetic hand.

Gained experience using CAD software and performing dynamic simulations in Inventor.
